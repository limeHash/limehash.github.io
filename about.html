Tentang Aplikasi

Sejarah

Pada mulanya, aplikasi ini dibangun untuk memenuhi kebutuhan sistem informasi salah satu rumah sakit swasta yang baru beroperasi di kota Malang sekitar tahun 2019. Pada awal perkembangannya aplikasi ini di design berbasis desktop dan hanya untuk melayani pendaftaran pasien rawat jalan. Seiring berjalannya waktu, aplikasi ini dikembangkan dan disesuaikan dengan kebutuhan rumah sakit, yang saat ini telah diperbarui menjadi aplikasi berbasis web.


Versi

  • v.1.0 : Berbasis desktop, dibangun dengan bahasa Java (tidak dilanjutkan)
  • v.2.0 : Berbasis web, awal masa pengembangan untuk unit Pendaftaran, Farmasi, Kasir
  • v.2.1 : Penambahan unit rawat jalan, rawat inap.
  • v.2.2 : Penambahan unit laboratorium, radiologi.
  • v.2.3 : Rewrite untuk perubahan UI
  • v.2.4 : Meminimalkan penggunaan resource, update fitur, penambahan ekstensi.
